Introduction
-------------
Elements of Consumtion (EoC) is an eco-visulaization of eletricity, water, and natural gas consumption within a home. Real-time fluid dynamics is use to create abstract, ever-chaning ambient art. For more infomation you can read the paper listed in the Citations Section below.

To compile/run EoC you need to have a Processing 1.5.1 installed on your computer. You can 
download Proccesing from: http://processing.org/. 
The most up-to-date source code can always be found on GitHub at: https://github.com/smakonin/EoC/.
